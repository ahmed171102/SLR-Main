# Web Frontend - SLR System

This directory will contain the React application for user interaction.

## Planned Structure (Phase 3)

- `src/App.tsx`: Main React component.
- `src/components/Camera.tsx`: Camera input with MediaPipe.
- `src/components/HandOverlay.tsx`: Interactive hand landmark visualization.
- `src/services/api.ts`: API integration with Backend.

## Technologies
- **React**: UI library basically.
- **MediaPipe**: Client-side hand tracking.
- **WebSocket**: Real-time communication for predictions.

See `../Deployment/docs/FULL_DEPLOYMENT_PLAN.md` for full instructions.
